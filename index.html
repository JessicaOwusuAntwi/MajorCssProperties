CSS stands for Cascading Style Sheet. It is used for website layout in design and used alongside HTML. CSS is a styling language.

There are three types or methods of CSS. There's In-line Css,Internal CssandExternal Css.

In-line CSSis added in the opening tag of anHTMLelement as a style attribute.

Internal CSS is added to your html file using a style tag in the head element just after the title tag.

External CSS is a little different. It is an external Css file that is linked to your html file. The link is added in the head of an html file.

There are also Css Selectors used to select html elements you want to style. We have element selectors, class selectors, id selectors, just to name a few.

There also is Css specificity, cascade hierachy and inheritance. These concepts control what Css applies to what element

Marginsare used to create space around elements, outside of any defined borders.It can also accept up to four values fortop, right, bottomandleftmargins.
